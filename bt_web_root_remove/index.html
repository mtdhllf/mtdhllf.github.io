宝塔-网站文件夹无法删除

2022-07-10 / 1 min read

宝塔在Linux上建站的时候会生成一个.user.ini文件，而且这个文件不能被删除，即使是root用户也删除不了。

使用下面命令的时候，发现文件多了一个i属性。

lsattr .user.ini

经过查证，i代表不得任意更动文件或目录，是为了防止重要文件被误删除，因此root也删除不了。
如果真想要删除文件，可以使用

chattr -i .user.ini

去掉文件的i属性，这样就可以删除该文件了。
